# Avatar

ユーザーアバター Atom。画像表示・頭文字フォールバック・4サイズに対応。

## 使用元プロジェクト

- なし（新規追加）

## Props

| Prop | 型 | デフォルト | 説明 |
|---|---|---|---|
| `src` | `string` | — | 画像URL。指定がない場合は `name` の頭文字を表示 |
| `alt` | `string` | — | 画像の代替テキスト。未指定時は `aria-hidden="true"` を付与 |
| `name` | `string` | — | 頭文字フォールバック用の名前（先頭1文字を大文字表示） |
| `size` | `'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` | サイズ（sm=24px / md=32px / lg=40px / xl=56px） |

## 移植時の注意

- Next.js で使う場合はファイル先頭に `'use client'` を追加する
- 意味のあるアバター（ユーザーを識別する文脈）では `alt` または `name` を必ず指定する
- `src` なし・`name` なしの場合は `?` が表示される
- 背景色は `--color-accent`、文字色は `--color-accent-text` を使用しているため、デザイントークンの差し替えで色を変更できる
