# ErrorMessage

エラーメッセージ表示 Atom。`role="alert"` によりスクリーンリーダーで即時読み上げされる。フォームバリデーションエラーの表示に適している。

## 使用元プロジェクト

- なし（新規追加）

## Props

| Prop | 型 | デフォルト | 説明 |
|---|---|---|---|
| `children` | `React.ReactNode` | — | エラーメッセージのコンテンツ（必須） |
| `id` | `string` | — | スクリーンリーダー連携用ID。フォームの `aria-describedby` から参照する |

## 移植時の注意

- Next.js で使う場合はファイル先頭に `'use client'` を追加する
- フォーム入力欄と連携する場合は `id` を指定し、入力欄の `aria-describedby` に同じ値を設定する
- 入力欄には `aria-invalid="true"` も合わせて付与することを推奨する
- `role="alert"` により DOM に追加された瞬間にスクリーンリーダーが読み上げるため、エラー発生時に動的にレンダリングする使い方に適している
- 警告アイコンには `aria-hidden="true"` を付与しているため、スクリーンリーダーはテキストのみを読み上げる
